# 🌍 旅人日常 Traveler's Daily Life

## 📌 專案介紹
「旅人日常」是一個以旅遊為主題的網站，提供全球各地旅遊行程資訊，讓使用者能夠快速瀏覽並選擇適合自己的旅遊方案。

本網站以簡潔的介面設計與清楚的分類，呈現亞洲、歐洲、美洲及非洲等地區的熱門行程，並提供詳細的每日行程介紹與搜尋功能，提升使用者體驗。

---

## 🎯 專案特色

- 🌏 多地區旅遊分類（亞洲 / 歐洲 / 美洲 / 非洲）
- 🔍 行程搜尋功能（可依目的地、日期篩選）
- 📅 詳細每日行程介紹
- 🧭 導覽列固定設計（方便快速切換頁面）
- 🖼️ 圖片搭配文字，提升視覺體驗
- 🧾 關於我們頁面（公司簡介與理念）
